# Open-Json-Api

This is a live HEROKU Api for Use.
## Installation

Go to  [Json-Api](https://open-json-api.herokuapp.com/) for easy access


Adapted from [typicode/jsno-server](https://github.com/typicode/json-server)  



## Usage

```
GET    /employees
GET    /employees/{id}
POST   /employees
PUT    /employees/{id}
PATCH  /employees/{id}
DELETE /employees/{id}
```
